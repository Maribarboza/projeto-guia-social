# Guia Social 🌍

## 📌 Sobre o projeto
O Guia Social é uma plataforma que ajuda pessoas a encontrar projetos sociais, ONGs e iniciativas próximas.

## 🚀 Funcionalidades
- Listagem de projetos sociais
- Busca por categoria
- Interface simples e intuitiva

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## ▶️ Como rodar
1. Clone o repositório
2. Abra o index.html
